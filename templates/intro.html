Base-In-a-Case

Because it should be easier to sync Google App Engine and Freebase.


This demo app stores a table of users and a list of topics for each user, and syncs these values with a Freebase Base. This allows users and developers to view and edit the app's data through Freebase.
The front-end uses the Freebase Suggest javascript library to add topics to and retrieve topics from Freebase.
